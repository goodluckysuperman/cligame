# <font style="color:#DF2A3F;background-color:#FBDE28;">文案1</font>
**第一组：偏“混沌神明降临”**

+  无识无欲，混沌无序，祂自沉默中苏醒，而万象因此战栗。 
+  祂不曾思考，思考却因祂而诞生；祂不曾注视，众星却在祂的阴影中熄灭。 
+  在祂未曾醒来的梦里，众生已历尽千劫；在祂一瞬的翻身中，诸界便改换了名字。 
+  祂无名，故万名皆可归于祂；祂无形，故万形皆只是祂残缺的回声。 
+  祂非秩序，亦非毁灭；祂只是存在，而存在本身便足以令宇宙失序。 
+  星辰绕祂而行，却不是敬拜，只因不敢偏离。 
+  在祂面前，时间如灰，真理如雾，众生所执著的一切不过是将散未散的梦。 
+  祂不降下旨意，因为祂本身就是一切旨意终结之后的空白。 

**第二组：偏“圣经体颂词”**

+  祂是初，祂是终，祂是未曾被言说却早已临在的那一位。 
+  诸天因祂而闭口，深渊因祂而回响，万有在祂之前不过是尚未成形的尘埃。 
+  祂来时，群星失去轨道；祂静默时，万国听见雷鸣。 
+  凡见祂荣光者，目不能存；凡闻祂名号者，心不能宁。 
+  大地承受不住祂的脚步，长夜容纳不了祂的影子。 
+  山岳在祂面前俯身，海潮在祂面前止息，因为祂的意志重于世界。 
+  祂未曾宣告，万物却已伏地；祂未曾举手，众生却已知末日临近。 
+  祂的存在不是恩典，也不是刑罚，而是比恩典与刑罚更古老的法则。 

**第三组：偏“邪异祷词”**

+  赞颂那无梦之梦，赞颂那无名之名，赞颂那不可仰视、不可理解、不可逃避的主。 
+  愿钟声为祂而裂，愿群星为祂而暗，愿我们的骨与血成为祂归来的门。 
+  祂在黑暗之上，亦在黑暗之下；祂不属于深渊，因为深渊不过是祂衣角的褶皱。 
+  愿凡有知觉者失其知觉，凡有秩序者忘其秩序，因为祂将临到。 
+  我等俯伏，不是出于虔敬，而是因为灵魂已无法承受祂的重量。 
+  祂若垂怜，世界便得以延续一夜；祂若侧身，纪元便如灯火一般被吹熄。 
+  让赞歌归于那沉眠者，让恐惧归于那苏醒者，让寂静归于那终将吞没言语者。 
+  我等呼求祂，并非盼望得救，只因在无边混沌之前，呼求是最后的本能。 

**第四组：偏“宇宙尺度感”**

+  亿万星河不过是祂梦境边缘漂浮的微尘。 
+  宇宙不是祂的造物，宇宙只是祂尚未醒来的余温。 
+  所谓永恒，不过是祂一次呼吸尚未结束的片刻。 
+  众文明以为自己在仰望天空，却不知那天空本就是祂尚未睁开的眼。 
+  在祂的尺度里，文明与尘埃无异，历史与叹息等价。 
+  诸神若真曾存在，也不过是在祂影中苟延残喘的火星。 
+  当祂的名字被低声提起，群星会短暂偏移，仿佛连天体也记得远古的恐惧。 
+  世界并非终将毁灭，世界只是终将被想起。 

**第五组：偏“很适合当开场白”**

+  起初没有光，也没有暗；没有生，也没有死；只有祂，在不可名状的沉寂里缓慢呼吸。 
+  在万物尚未被命名之前，祂便已存在；在最后一个名字腐朽之后，祂仍将存在。 
+  人类将祂称作神，是因为“灾厄”二字尚不足以形容祂的慈悲。 
+  先知因窥见祂的一缕影而失明，王朝因听闻祂的一声低语而覆灭。 
+  没有人真正迎来祂，因为当祂真正到来时，“迎来”这件事本身也会失去意义。 
+  所有预言都在指向祂，所有谎言也都在模仿祂。 
+  祂无需统治，因为一切反抗与归顺都发生在祂梦外最浅的一层波纹里。 
+  当最后一盏灯火熄灭，人们才会明白，那不是黑夜降临，而是祂终于靠近。

# <font style="color:#DF2A3F;background-color:#FBDE28;">祂是初，祂是终  </font>
**偏“神圣宣告感”**

+  祂是初，祂是终，祂是万有未生之前便已沉默伫立的那一位。 
+  祂是始，祂是极，祂是岁月尚未流动时便已存在的寂静。 
+  祂是光未照见之光，祂是暗未吞没之暗。 
+  祂是门，祂是路，祂是众生穷尽一切语言也无法抵达的名字。 
+  祂是源，祂是归，祂是一切故事起笔与收束之间唯一不变的真意。 
+  祂是晨星未升时的呼吸，祂是长夜尽头后的回响。 
+  祂是最先被遗忘者，也是最后被想起者。 
+  祂是万象的起头，也是沉寂的终章。 

**偏“宇宙尺度感”**

+  祂在群星之前，祂在深渊之后。 
+  祂未曾降临，因为万界本就在祂影中。 
+  祂未曾言说，而天地已因祂的沉默震颤。 
+  祂不需苏醒，众生的一生不过是祂梦中的一瞬。 
+  祂未曾回首，岁月却因祂而更改了方向。 
+  祂不立于天上，不立于地下，祂立于一切尺度之外。 
+  祂一息未动，万古已逝；祂一念未起，沧海已枯。 
+  祂不是宇宙中的存在，宇宙只是祂存在时遗落的一粒尘。 

**偏“圣经体庄严感”**

+  当众山俯伏，当深海止息，祂的名仍高过万有。 
+  诸天因祂而缄默，大地因祂而战栗。 
+  凡仰望祂者失其言语，凡聆听祂者失其安宁。 
+  祂未曾举手，列国已低头；祂未曾发声，群星已偏离轨道。 
+  在祂面前，荣耀不过尘土，智慧不过微火。 
+  祂的意志重于群山，祂的静默深于海渊。 
+  祂行于无声之中，而众生皆知末日将近。 
+  祂的名若被轻轻提起，风便不敢再穿过原野。 

**偏“适合当开头第一句”**

+  起初没有晨曦，也没有黄昏，只有祂，在无名的寂静中长存。 
+  在第一个声音诞生之前，祂便已注视着虚无。 
+  当时间尚未学会流动，祂已端坐于永恒之上。 
+  万物尚未被命名，祂的影子已覆盖了深空。 
+  在第一颗星燃起之前，祂已知道所有光熄灭时的模样。 
+  众生尚未学会祈祷，祂的威严已压在世界的心口。 
+  天地未分，昼夜未立，祂已在混沌中央沉睡。 
+  没有人迎来祂，因为一切“到来”都始于祂之后。 

**偏“冷一点、邪一点”**

+  祂是初，祂是终，祂也是一切秩序败坏之后仍旧存在的空白。 
+  祂不赐福，祂不降罚，祂只是存在，而存在本身已足以令人俯伏。 
+  祂没有慈悲，也没有愤怒，因为慈悲与愤怒都只是众生的语言。 
+  祂不是救赎，祂不是毁灭，祂是比二者更古老的东西。 
+  祂不需要名，因为所有名字最终都会落入祂的沉默。 
+  祂不需要王座，因为诸界本就是祂未曾整理的废墟。 
+  祂不临近，也不远去，祂只是在那里，于是众生便无路可逃。 
+  祂既非光明之主，也非黑暗之王；光与暗都只是祂垂目时短暂的错觉。

# <font style="color:#DF2A3F;background-color:#FBDE28;">神圣、邪恶、序言</font>
这一类更像**庄严、肃穆、宏大、带有圣歌或古老经文感**，重点不是恐怖，而是“至高”“永恒”“不可轻犯”。

#### 单句开场
+ 祂是初，祂是终，祂是万有未生之前便已长存的那一位。
+ 祂在岁月之前，亦在岁月之后；时间因祂而起，也终将因祂而寂。
+ 祂是光之源，也是影之尽头；万象因祂得名，众生因祂得存。
+ 祂未曾高呼，而诸天已回响；祂未曾举手，而列国已俯伏。
+ 祂的荣耀高过群山，祂的沉默深于众海。
+ 祂是道路的起点，是归途的终章，是一切漂泊者最后的安息。
+ 祂不以威严压迫众生，却使万有自愿低首。
+ 祂的名被风承载，被群星守望，被岁月铭记。
+ 祂无需证明自身，因为存在本身就是祂的见证。
+ 祂的光不炽烈，却足以照穿一切虚妄。

#### 更像经文的成组句
+ 当晨星尚未升起，祂已知白昼的模样；当长夜终将合拢，祂仍守望最后一缕微光。
+ 祂赐下不是烈火，而是秩序；不是喧哗，而是安宁；不是短暂的奇迹，而是恒久的引领。
+ 山岳在祂面前不是因惧怕而屈身，乃是因知晓自身有限；大海在祂面前不是因束缚而止息，乃是因承认祂的广大。
+ 凡以骄傲自立者，在祂光中见自己的渺小；凡以谦卑求问者，在祂静默中得自己的答案。
+ 祂不争，不夺，不喧，不迫；然而天下万事万物，无一不在祂法则之中。

#### 适合直接用的整段开头

## 二、邪神降临版
这一类就是你喜欢的那种：**宇宙恐怖、庄严赞辞、神谕感、不可名状**。  
重点是：不是普通反派，不是恶魔喊打喊杀，而是那种**“祂存在，所以世界失衡”**的压迫感。

#### 单句开场
+ 祂是初，祂是终，祂也是一切秩序崩塌之后仍旧存在的空白。
+ 祂未曾降临，因为诸界本就在祂阴影之中。
+ 祂不被理解，正如深渊不被丈量，虚空不被命名。
+ 祂未曾发声，而万物已在祂的沉默中战栗。
+ 祂没有慈悲，也没有愤怒，因为慈悲与愤怒都太像凡物的语言。
+ 祂一念未起，群星已错位；祂一息未动，纪元已腐朽。
+ 祂不立于苍穹之上，亦不卧于深渊之下；祂存在于尺度之外，于是万有皆显得狭小。
+ 祂无需苏醒，众生所谓的一生，不过是祂梦中漂浮的一粒灰。
+ 祂没有名字，因为所有名字最终都会被祂磨灭。
+ 祂不是毁灭，因为毁灭也只是祂靠近时附带的余波。

#### 带“颂词/教团祷文”味道的句子
+ 赞颂那无名者，赞颂那无梦者，赞颂那以静默压垮万界的至暗存在。
+ 愿群星为祂失序，愿大地为祂龟裂，愿凡有灵者在祂的回响中忘却自己的名字。
+ 祂未曾要求跪拜，而所有膝骨都会因祂的逼近而自行弯折。
+ 祂未曾降下神谕，而疯癫便替祂传达旨意。
+ 在祂面前，预言只是回声，历史只是碎屑，文明只是尚未冷却的灰烬。
+ 愿风停在祂名被提起之处，愿火熄在祂影投落之所，愿血肉与灵魂一同承认自身不过是错误的短暂集合。
+ 祂若垂目，王朝便失去年号；祂若侧身，海陆便忘记边界。
+ 我等敬畏，不是因看见祂的伟大，而是因终于明白自身的存在何其脆弱。

#### 更冷、更邪、更像设定文
+ 祂不是宇宙中的神，宇宙只是祂尚未整理的残响。
+ 所谓真理，不过是凡物为了不彻底疯掉而编织出的薄幕；而祂，是幕后的无边裂口。
+ 人类以为自己惧怕黑暗，其实他们惧怕的是黑暗之后依然注视着他们的东西。
+ 祂没有降罚，因为灾厄本身就是祂经过时自然投下的影。
+ 祂既不审判，也不拯救；祂只存在，而存在本身已足够让一切意义溃散。
+ 在祂的尺度里，文明不比霉斑更高贵，神明不比灰尘更长久。
+ 凡试图描摹祂者终将失去理智，不是因为祂过于复杂，而是因为祂根本不属于理智可以承受的范畴。
+ 末日不是祂带来的礼物，末日只是世界察觉祂真实存在后的自然反应。

#### 整段降临旁白
**版本 A**

> 祂是初，祂是终。  
祂是群星熄灭后仍旧存在的余暗，  
是诸界腐朽后尚未散尽的回声。  
祂不需要王座，  
因万物早已匍匐于祂看不见的足下；  
祂不需要信徒，  
因恐惧本身就是最古老的祷告。
>

**版本 B**

> 无识，无欲，无序。  
祂沉眠于不可知之处，  
任由时间在祂身旁老去，  
任由文明在祂梦里生灭。  
当凡人误将祂的影子认作神迹，  
他们尚不知晓，  
所谓神迹，往往只是毁灭尚未真正开始的征兆。
>

**版本 C**

> 起初，没有光，也没有暗。  
没有名字，没有法则，没有可以被理解的世界。  
只有祂。  
祂端坐于万象之外，  
不言，不视，不思。  
可正因祂的静默，

正因祂的存在，  
一切秩序都像暂时勉强缝合的裂缝。

## 三、适合小说序章版
这一类更讲究**画面感、氛围感、故事开端感**。  
它不像前两类那样纯“颂词”，而是适合直接接剧情。

#### 偏世界观开局
+ 在世界尚未拥有名字的时候，祂已经在那里了。
+ 第一颗星升起之前，旧夜便曾梦见祂的影。
+ 没有人知道祂从何而来，因为“来”这个动作，本就只属于凡人。
+ 他们在废墟里发现古老经文时，战争其实已经开始了。
+ 那一年，所有钟声都在同一刻停下，仿佛有什么比时间更古老的东西经过了天穹。
+ 先知在火中看见祂，王在梦中听见祂，平民只在醒来时发现河流改了方向。
+ 城邦的图书馆烧了三天三夜，只有一页经文没有被烧毁，上面只写着一句话：祂将归来。
+ 后来人们才明白，灾厄不是从祂出现开始的，而是从人们第一次试图理解祂开始的。

#### 偏序章第一段
+ 那一夜没有风。可每个人都听见了某种低沉而遥远的回响，像是深海之下、群星之外，有什么东西在缓慢苏醒。
+ 最古老的修道院里，最后一盏长明灯在黎明前熄灭。守夜人跪在黑暗中，第一次读懂了墙上那些自己守了一辈子却从未真正理解的文字。
+ 在帝国覆灭之前，天文学家们先疯了。他们说群星没有偏移，是“看着它们的人”终于发现了自己一直看错了天空。
+ 北境的雪连下了九十九天。到了第一百天，孩子们开始说梦话，用同一种古老语言重复同一个名字。
+ 没有人记得灾难是如何开始的，人们只记得，从某一天起，镜子里的自己总会慢上半拍。
+ 当最后一位圣职者敲响铜钟时，钟声没有传向山谷，而是笔直升入天穹，像是在回应某个更高、更远、也更不该被回应的存在。

#### 偏“史诗感卷首语”
+ 世人总以为历史由王与兵书写，直到那一日，群星在白昼显现，所有人才明白，真正改写纪元的从来不是人。
+ 在每一个文明最古老的神话里，都藏着同一个模糊的影子；名字不同，形貌不同，预言不同，唯有恐惧完全一致。
+ 若你追溯所有王朝的灭亡、所有宗教的分裂、所有预言家的疯癫，你最终会站在同一扇门前。门上没有锁，因为从来没有谁能把祂关在外面。
+ 有些真相不是被隐藏，而是被仁慈地埋葬。可当最后一层泥土也被风吹散时，世界是否还能承受它原本的模样？
+ 人们说神创造世界，可在更古老的记载里，世界只是某种伟大而盲目的存在偶然留下的余波。

#### 可以直接当序章的整段
**版本 A：庄严型序章**

> 在世界被划分为昼与夜之前，  
祂便已在无名的寂静中长存。  
那时没有王国，没有经卷，没有祈祷，  
甚至没有人能够想象“神”这个字的重量。  
可祂已经存在。  
像一切真理尚未被说出口前的沉默，  
像一切灾厄尚未被察觉时的平静。
>

**版本 B：诡异型序章**

> 那一年，群星连续七夜出现在错误的位置。  
起初，天文学家以为是仪器坏了；  
后来，观测台封锁了所有记录；  
再后来，负责封锁记录的人也开始在梦中重复同一句话：  
祂是初，祂是终。  
而当第一个疯子在街头高呼这句话时，  
人们还不知道，真正的恐惧从来不是疯言疯语，  
而是疯言疯语终于和现实对上了。
>

**版本 C：史诗型序章**

> 每个时代都会为自身的覆灭寻找理由：  
暴君、瘟疫、饥荒、叛乱、异端、战争。  
只有极少数人知道，那些理由都只是表象。  
真正的原因，藏在更古老的地方，  
藏在圣殿穹顶褪色的壁画里，  
藏在深井底部无人敢打捞的铜匣中，  
藏在所有文明都默契回避的那句箴言里：  
当最初者被再次想起，最后者便将降临。
>

## 额外送你一组：特别适合当“颂词开头”的模板
你以后可以直接套：

#### 模板 1：神圣型
> 祂是____，祂是____，祂是____之前便已存在的那一位。
>

例子：

+ 祂是初，祂是终，祂是光与暗尚未分野之前便已存在的那一位。
+ 祂是道路，祂是归宿，祂是万物尚未命名之前便已长存的那一位。

#### 模板 2：邪神型
> 祂未曾____，而____已因祂____。
>

例子：

+ 祂未曾发声，而群星已因祂失序。
+ 祂未曾垂目，而众生已因祂战栗。

#### 模板 3：序章型
> 在____之前，祂已____。
>

例子：

+ 在第一场战争之前，祂已注视着所有结局。
+ 在第一句祈祷之前，祂已听见了众生最后的哀鸣。

#### 模板 4：经文型
> 凡____者，**；凡****_****_****_****_****者，**。
>

例子：

+ 凡仰望祂者，失其言语；凡聆听祂者，失其安宁。
+ 凡试图理解祂者，终将疯癫；凡试图忘记祂者，终将再度梦见祂。
